.. _agarose:

5% agarose (w/v)
================

**Requires**

* Agarose powder
* Milli-Q water/distilled water
* Empty bottle (50 mL)
 
**Procedure**

#. Dissolve 5 g of agarose powder in 100 mL distilled water/Milli-Q water. 
#. Label.

    Label example: :code:`5% Agarose, name initials, date prepared`

#. Agarose can be stored on bench.
